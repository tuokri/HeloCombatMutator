# Helicopter Combat Mutator

A mod for Rising Storm 2: Vietnam. Makes all players in both teams pilots.
Adds red helicopters for North Vietnam.

## Unique Features

- Heat seeking MANPADS Strela-2.
- Rocket launching machine gun (DShK).
- Helicopter flares.
- Parachutes.
